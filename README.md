# 🎓 Student Grade Management System

A simple yet powerful command-line interface (CLI) application for managing student grades using CSV file storage.

## ✨ Features

- ➕ **Add Students**: Add new students with their names and grades
- 👥 **View All Students**: Display all students with their grades in a formatted table
- 📊 **Grade Statistics**: Calculate and display average grades
- 💾 **Persistent Storage**: Data is automatically saved to a CSV file
- 🛡️ **Error Handling**: Robust error handling for file operations and invalid inputs

## 🚀 Installation

### Prerequisites
- Python 3.6 or higher
- No additional libraries required (uses only built-in modules)
